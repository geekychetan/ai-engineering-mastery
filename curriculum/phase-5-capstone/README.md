# Phase 5 — Capstone, Portfolio & Interview Prep (Weeks 12-15)

**Goal:** build one impressive, well-documented capstone that ties everything together, then package yourself for the job hunt and get interview-ready.

---

## Weeks 12-13 — Capstone build
A production-shaped LLM app combining **RAG + an agent + an eval suite**, containerized and **deployed on AWS**.

Anchor domain (plays to your strengths — pick one):
- **LLM-powered QA/testing assistant** — e.g., generates test cases from requirements, triages bug reports, or reviews test coverage. Leverages the manual-QA background directly.
- **RAG assistant with a rigorous eval harness** — a domain doc assistant whose standout feature is *how well it's evaluated*.

Must-haves:
- A **problem brief** (`templates/project-brief-template.md`) written first.
- An **architecture diagram** (Excalidraw).
- A thorough **README**: problem, approach, how to run, **how it's evaluated**, cost/latency notes, limitations.
- The eval suite with real numbers (before/after prompts, judge scores).

## Week 14 — Portfolio & personal brand
- Clean up GitHub: profile README, **pinned repos**, good project READMEs.
- **Resume** — frame MBA + Marketing/Salesforce + QA → AI as a *pivot strength*, not a gap.
- **LinkedIn** — headline, About, projects; start posting build notes.

## Week 15 — Interview prep + open source
- **LLM system design:** design a RAG system on a whiteboard; hallucination mitigation; evals; cost/latency/quality tradeoffs.
- Common junior AI/LLM interview questions (technical + applied).
- **Behavioral / career-pivot narrative** — tell the story crisply.
- Mock interviews.
- Stretch: open **one real open-source PR** to an AI project (huge credibility signal).

---

## Exit criteria — you are job-ready when
- [ ] Capstone is deployed, documented, and has a real eval suite.
- [ ] GitHub profile + pinned repos look professional; contribution graph is green.
- [ ] Resume + LinkedIn tell the pivot story convincingly.
- [ ] You can whiteboard a RAG system and discuss tradeoffs out loud.
- [ ] You can answer "how do you know your LLM app works?" with specifics.
- [ ] (Stretch) One open-source PR opened.
