# Resources — "Learn it here"

Curated, mostly-free resources per phase. **Free path is complete on its own**; paid options are optional accelerators.

> Note on links: canonical course/doc landing pages are linked directly. For **specific YouTube videos and Udemy courses** the exact title + creator is given so you can search — those URLs change often, so search the title rather than trusting a pasted link.

---

## Phase 1 — Python + Git Foundations

**Python (free)**
- **CS50P — Introduction to Programming with Python** (Harvard): https://cs50.harvard.edu/python — *primary course, runs in parallel.*
- YouTube — **Corey Schafer**, "Python Tutorial for Beginners" playlist.
- YouTube — **freeCodeCamp**, "Python for Beginners" (4-hr).
- **Python for Everybody** (Dr. Chuck), free: https://www.py4e.com

**Python (paid, optional)**
- Udemy — *"100 Days of Code: The Complete Python Pro Bootcamp"* — **Angela Yu** (do days 1–40).

**Git & GitHub (free)**
- **Pro Git** book: https://git-scm.com/book
- **Atlassian Git tutorials**: https://www.atlassian.com/git/tutorials
- **GitHub Skills** (interactive): https://skills.github.com
- YouTube — **freeCodeCamp**, "Git and GitHub for Beginners".

**Testing / pytest (free)**
- pytest docs: https://docs.pytest.org
- YouTube — **ArjanCodes** and **Corey Schafer** unit-testing videos.
