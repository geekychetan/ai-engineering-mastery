# Projects

Builds grow with the phases. **Every project starts with a brief** (`../templates/project-brief-template.md`) and ends with **how it was evaluated**.

- `small/` — Phase 1–2 warm-ups (CLI tools, API scripts, one sklearn model).
- `intermediate/` — Phase 3–4 (RAG Q&A, tool-using agent, chatbot + eval suite, deployed app).
- `capstone/` — Phase 5 (RAG + agent + eval suite, deployed on AWS).
