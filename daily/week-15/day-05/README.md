## Homework (branch → PR)
1. [ ] Branch: `git switch -c week15-day05-mock-interview`.
2. [ ] Run a ~45–60 min mock: 2–3 technical questions, one applied/case, one whiteboard RAG design, and the pivot story + one behavioral. Do it live and recorded — a friend as interviewer is ideal; otherwise self-drive with a timer.
3. [ ] Watch the recording back and score yourself in `notes/interview/mock-review.md`: clarity, did you ask clarifying questions, did you name eval plans, filler words, body language, time management.
4. [ ] Extract the top 3 weaknesses and a concrete fix for each (e.g. "rambled on RAG — rehearse a 60s version").
5. [ ] Re-do the ONE question you did worst on and re-record it; compare.
6. [ ] Push, PR with `Closes #<issue>`, self-review, merge.
